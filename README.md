# Day Trader NL

Een moderne, interactieve Day Trading simulatie-app gebouwd met Flutter, speciaal ontworpen voor de Nederlandse markt.

## Functionaliteiten

* **Markten Dashboard:** Real-time (gesimuleerde) weergave van actuele aandelenkoersen.
* **Interactieve Grafieken:** Professionele koersgrafieken dankzij `fl_chart`.
* **Handelsacties:** Kopen en verkopen via pop-up dialogen.
* **Watchlist:** Houd je favoriete aandelen en crypto's (zoals ASML, Apple, Tesla en Bitcoin) in de gaten.
* **Portfolio Weergave:** Bekijk je huidige bezittingen en saldo's (in ontwikkeling).
* **Donkere Modus:** Een strak, donker kleurenschema voor een moderne trader-ervaring.

## Tech Stack

* **Flutter:** Core framework voor een vloeiende, native cross-platform ervaring.
* **fl_chart:** Voor het renderen van de dynamische, visuele handelsgrafieken.
* **google_fonts:** Voor een strakke, moderne typografie (Inter).

## Installatie & Uitvoeren

Zorg ervoor dat je de Flutter SDK (versie 3.7.2 of nieuwer) hebt geïnstalleerd.

1. Kloon of download dit project.
2. Run `flutter pub get` om alle dependencies (inclusief `fl_chart` en `google_fonts`) op te halen.
3. Start de app op een emulator of verbonden apparaat via `flutter run`.
